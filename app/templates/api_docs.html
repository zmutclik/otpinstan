1. Order OTP GET /getotp

Memesan nomor HP baru untuk menerima OTP.

URL:
{{ base_url }}/getotp?service=go&country=6
Parameter:
Param Deskripsi Contoh
service Kode layanan wa, tg, ig, go, sh
country ID negara 6 (Indonesia)
Response Success:
{
  "success": true,
  "order_id": "S5-123456789",
  "phone": "628123456789",
  "service": "go",
  "country": 6,
  "price": 512,
  "status": "pending"
}

2. Cek OTP GET /chekotp

Cek apakah OTP sudah diterima untuk nomor HP tertentu.

URL:
{{ base_url }}/chekotp?phone=628123456789
Response:
{
  "success": true,
  "order_id": "S5-123456789",
  "phone": "628123456789",
  "status": "received",
  "otp": "957418",
  "otp_updated_at": "2026-07-01T10:15:00"
}

3. Status Sukses GET /status/sukses/{phone}

Laporkan bahwa nomor HP berhasil digunakan.

URL:
{{ base_url }}/status/sukses/628123456789

4. Status Gagal GET /status/gagal/{phone}

Laporkan bahwa nomor HP tidak bisa digunakan.

URL:
{{ base_url }}/status/gagal/628123456789
